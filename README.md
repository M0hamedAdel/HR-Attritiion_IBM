# HR-Attritiion_IBM
Employee Attrition Dashboard built with Python, Streamlit, Plotly, and Machine Learning. Analyzes employee turnover patterns, visualizes HR insights, and predicts attrition using a Random Forest model. Includes KPI metrics, interactive charts, department filtering, and feature importance analysis.
